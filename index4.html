Dependability Requirements

For Automatic Train Protection System (ATP)

Question Statement

Given is the basic structure and functioning of an Automatic Train Protection System and we are asked to the following:

  1. Task 1 - Risk Driven Specification
    Apply Risk-driven specification approach and provide a set of dependability requirements for the ATP. In particular:
    • Describe the risks
    • Assess the seriousness of each risk
    • Decompose the risks to discover their potential root causes
    • Define the dependability requirements: how each risk must be taken into eliminated or reduced when the system is designed.
  2. Task 2 - System Design
    Design the ATP and provide a Design Document reporting the following info:
    1. Outline of the design:
      • High-level description of the design
    2. Major design issues:
      • Important issues to be resolved (w.r.t. the requirements defined in Task 1)
      • All alternatives you considered, the final decision and the rationale for it
    3. Design details:
      • Software Architecture
        • Use Case Diagram
        • UML Component Diagram
        • Deployment Diagram
